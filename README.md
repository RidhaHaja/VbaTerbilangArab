# VbaTerbilangArab
VBA Terbilang Bahasa Arab pro+

PENGANTAR
Akhiri Debat "Muzakkar-Muannas" di Raport dengan Sekali Klik

Siapa yang tidak pusing saat mengisi raport pondok, lalu berdebat dengan rekan sejawat hanya karena bingung angka ini harusnya ditulis pakai Muzakkar (laki-laki) atau Muannas (perempuan)? Belum lagi urusan Irab (perubahan akhir kata) yang harus benar secara kaidah Nahwu.

Masalah ini klasik, tapi cukup menyita waktu. Berangkat dari kegelisahan itulah, saya mengembangkan kode Terbilang Arab Pro+ ini. Dibuat khusus untuk kebutuhan Madrasah dan Pondok Pesantren, tool ini bukan sekadar mengubah angka jadi kata. Ia sudah "paham" aturan gender, tahu bedanya urutan kelas dengan jumlah benda, dan mendukung format mata uang. Selamat tinggal perdebatan manual, selamat datang otomatisasi yang syar'i secara kaidah!

**Fitur Utama Kode**

**Irab (Tata Bahasa)**: Mendukung perubahan akhiran kata (seperti Alfani menjadi Alfaini) yang sangat penting untuk konteks formal atau keuangan.

**Gender (Muzakkar/Muannas)**: Menangani aturan "angka berlawanan" (angka 3-9 menggunakan gender yang berlawanan dengan benda yang dihitung).

**Gaya Penulisan**: Pilihan antara penulisan klasik (dipisah dengan Alif) atau modern (disambung) untuk angka ratusan.

**Mode Eja**: Selain mengubah angka menjadi kalimat, tersedia mode untuk mengeja angka per digit.
